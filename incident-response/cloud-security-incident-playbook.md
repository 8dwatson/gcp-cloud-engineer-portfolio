# Cloud Security Incident Response Playbook

## Scenario
Detection of suspicious privilege escalation in cloud environment.

---

## Phase 1: Identification

- Review audit logs
- Identify affected account
- Confirm unauthorized role change

---

## Phase 2: Containment

- Revoke elevated permissions
- Disable affected account
- Rotate credentials

---

## Phase 3: Investigation

- Analyze logs timeline
- Check data access history
- Identify root cause

---

## Phase 4: Recovery

- Restore proper IAM configuration
- Reinstate secure access
- Apply preventive controls

---

## Lessons Learned

- Enforce MFA
- Automate IAM alerts
- Conduct quarterly access reviews
