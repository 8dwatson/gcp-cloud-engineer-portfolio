# GCP Threat Model

## Assets

- Customer data
- Service accounts
- IAM policies
- Compute resources

---

## Threat Actors

- External attackers
- Malicious insiders
- Compromised credentials

---

## Attack Vectors

- Over-permissioned IAM roles
- Publicly exposed services
- Weak authentication
- Stolen API keys

---

## Mitigation Strategy

- Enforce least privilege
- Enable logging and monitoring
- Use private networking
- Enable MFA
- Monitor IAM policy changes

---

## Security Goal

Reduce attack surface and detect privilege misuse quickly.
