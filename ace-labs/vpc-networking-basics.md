# VPC Networking Basics (ACE Lab)

## Objective
Understand and configure basic VPC networking.

---

## Concepts Covered

- Create custom VPC
- Subnets and regions
- Firewall rules
- Private vs public IP
- VPC flow logs

---

## Security Considerations

- Restrict ingress traffic
- Avoid 0.0.0.0/0 on SSH
- Enable flow logs for visibility
